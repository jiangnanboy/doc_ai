#### About

这里将paddle中的ocr等模型转为onnx格式，并利用java版深度框架djl加载这些onnx模型进行推理预测尝试。

#### Getting started
onnx模型下载：

链接：https://pan.baidu.com/s/1Vcsn-VPTU_GHB_KU4pQFPg

项目启动后可以进入swagger进行接口使用 http://localhost:8080/swagger-ui/
